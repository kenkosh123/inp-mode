# inp-mode
Emacs major mode for ABAQUS inp file.

## Features
* Syntax highlighting

## Syntax highlighting

```inp
**COMMENT
*KEYWORD, OPTION1, OPTION2=VALUE2, ...
DATA1, DATA2, DATA3, ...
```

| Face      | Color       |
|-----------|-------------|
| **COMMENT | green       |
| *KEYWORD  | bold yellow |
| OPTION    | yellow      |
| VALUE     | cyan        |
| DATA      | white       |
| Comma (,) | white       |
| Equal (=) | white       |
